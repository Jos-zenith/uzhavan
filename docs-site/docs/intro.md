# VICT Documentation

This portal provides structured developer documentation for the VICT Digital Agriculture Platform.

## Sections

- **Installation & Setup**: local prerequisites, dependency installation, validation, run, and build.
- **SDK & Governance**: architecture overview for telemetry, offline support, and ROI workflows.
- **Policy-First Telemetry**: enforcement model and migration guidance.

## Start Here

1. Go to [Installation](installation)
2. Publish and install your SDK from npm via [SDK npm Installation](sdk-npm-installation)
3. Review [SDK Framework](sdk-framework)
4. Implement tracking using [Policy Enforcement](policy-enforcement)
5. Migrate existing telemetry with [Policy Migration](policy-migration)
